LBO Valuation — レバレッジが価値を変えるメカニズム LBO Valuation — How Leverage Transforms Value

レバレッジド・バイアウト（LBO）は、デットを使ったエクイティ・リターン増幅の仕組みである。PE投資家のIRR思考とDCFはいかに統合されるか。デット・キャパシティはどの変数に制約されるか。出口マルチプルの感応度がなぜ評価を支配するか。そしてEquity waterfallはどのようにキャッシュを分配するか。本稿はLBOバリュエーションを構成する理論的部品を解析する。 A leveraged buyout (LBO) is an equity return amplification mechanism built on debt. How does the PE investor's IRR thinking integrate with DCF valuation? What variables constrain debt capacity? Why does exit multiple sensitivity dominate the analysis? And how does the equity waterfall govern cash distribution? This article dissects the theoretical components of LBO valuation.

概要Summary

LBOは「他人のお金でエクイティ・リターンを増幅する」メカニズムであり、そのリターンの源泉は①デット返済によるエクイティ拡大、②EBITDA成長、③マルチプル拡張の3つに分解される。DCFとの統合はAPV（Adjusted Present Value）アプローチにより達成され、レバレッジによる節税効果が「なぜデットが企業価値を高めうるか」の理論的根拠を与える。実務においてはデット・キャパシティの制約と出口マルチプルの感応度が評価の大半を左右する。 An LBO is a mechanism that amplifies equity returns using other people's money. The three sources of return are: (1) debt paydown expanding the equity claim, (2) EBITDA growth, and (3) multiple expansion. Integration with DCF is achieved via the Adjusted Present Value (APV) approach, which provides the theoretical basis for why debt can enhance enterprise value through the interest tax shield. In practice, debt capacity constraints and exit multiple sensitivity dominate the analysis.

Valuation実務家への含意Implications for Practitioners
  • 出口マルチプルの感応度を先に作る — LBO評価において最もリターンを左右する変数は出口EV/EBITDAマルチプルである。エントリー価格の交渉前に、出口マルチプル × 出口EBITDA の感応度テーブルを作成し、許容できるエントリー価格の上限を逆算する規律が必須である Build exit multiple sensitivity first — The variable that most determines LBO returns is the exit EV/EBITDA multiple. Before negotiating entry price, construct a sensitivity table of exit multiple × exit EBITDA and back-solve for the maximum acceptable entry price. This discipline must precede, not follow, price discussions.
  • デット返済スケジュールはFCFで検証する — EBITDA倍率で概算したデット水準を、FCFプロジェクションで検証することが必須である。利払い後の実質FCFがゼロ近傍になるケースはリファイナンス・リスクを内包しており、モデル上のIRRより現実のリターンは低くなる Validate debt schedule against FCF projections — Any debt level sized by an EBITDA multiple must be stress-tested against FCF projections. If free cash flow after interest is near zero, the transaction carries refinancing risk and real-world returns will fall short of the modelled IRR.

Use the APV approach, or at minimum apply the Miles-Ezzell model to update WACC annually as leverage declines with debt repayment.

LBOの基本構造：デットがエクイティ・リターンを変える理由 LBO Fundamentals: Why Debt Transforms Equity Returns

レバレッジド・バイアウトとは、対象企業のキャッシュフローを担保にデットを調達し、エクイティ出資を最小化して企業を買収するストラクチャーである。PE（プライベート・エクイティ）ファンドがLBOを選好する理由は単純だ。同じ絶対リターンをより少ないエクイティ元本で達成することで、IRR（内部収益率）を大きく引き上げられるからだ。 A leveraged buyout is a structure in which debt is raised against the target company's cash flows, minimising equity investment to acquire a business. The reason PE funds prefer LBOs is straightforward: by achieving the same absolute return on a smaller equity base, IRR (internal rate of return) is dramatically amplified.

具体例で確認しよう。企業価値100で買収できる企業があるとする。5年後に企業価値が150になると仮定したとき、エクイティ100%で買うケースとデット60・エクイティ40で買うケースを比較する。 Consider a concrete example. A target with enterprise value 100 can be acquired. If enterprise value grows to 150 in five years, compare the all-equity case against a structure with 60 debt and 40 equity.

レバレッジ効果の比較（単位：億円）Leverage Effect Comparison (units: ¥bn)
全エクイティAll Equity LBO（D60/E40）LBO (D60/E40)
エントリー EVEntry EV 100 100
エントリー デットEntry Debt 0 60
エントリー エクイティEntry Equity 100 40

同じ企業価値の上昇（50）から、エクイティ・リターンに3倍以上の差が生じる。これがレバレッジの本質的な効果である。ただしこの計算は単純化されており、デット返済に使われたFCFはエクイティ投資家の手取りには含まれないことに注意が必要だ。 The same enterprise value increase (50) generates more than three times the equity return difference. This is the fundamental effect of leverage. Note, however, that this calculation is simplified — the FCF used to repay debt does not flow directly to equity investors and is the source of the equity value build-up.

リターンの3つの源泉：構造的な分解 Three Sources of Return: A Structural Decomposition

LBOのエクイティ・リターンは理論的に以下の3つの源泉に分解できる。実務ではこの分解がバリュー・クリエーション分析（Value Creation Analysis）として提示される。 LBO equity returns can theoretically be decomposed into three sources. In practice this decomposition is presented as a Value Creation Analysis.

  • 1
    デット返済（Debt Paydown）Debt Paydown — 事業から生み出すFCFでデットを返済することで、エクイティの取り分（EV − Net Debt）が拡大する。このコンポーネントはEBITDAとCapexの差、すなわちFCF生成能力に依存する。デット返済はリスク調整後リターンでは最も確実性が高い。 — Repaying debt with operating FCF expands the equity claim (EV − Net Debt). This component depends on the spread between EBITDA and capex — i.e., FCF generation capacity. Debt paydown is the highest-certainty component on a risk-adjusted basis.
  • 2
    EBITDA成長EBITDA Growth — 売上成長・マージン改善・コスト削減によりEBITDAを増やすことで、同じマルチプルでもEVが拡大する。PE投資家が「100日プラン」や「バリュークリエーション・プラン」と呼ぶオペレーショナル改善の主体はここにある。 — Growing EBITDA through revenue growth, margin improvement, and cost reduction expands EV at any given exit multiple. This is the domain of the "100-day plan" and "value creation plan" that PE investors pursue through operational improvement.
  • 3
    マルチプル拡張（Multiple Expansion）Multiple Expansion — エントリー時より高いEV/EBITDAマルチプルで出口を迎えることで、EBITDAの増分以上にEVが上昇する。PE業界では「マルチプル・アービトラージ」とも呼ぶ。市場環境の改善、事業品質の向上（プラットフォーム化・SaaS転換等）がドライバーだが、予測可能性は最も低い。 — Exiting at a higher EV/EBITDA multiple than entry causes EV to rise beyond the EBITDA increase alone. The industry calls this "multiple arbitrage." Drivers include market environment improvements and business quality enhancement (platformisation, SaaS conversion), but this component has the lowest predictability.

「我々は複数のリターン・ドライバーを持っているが、正直に言えば、過去10年のPEリターンの大部分はマルチプル拡張が説明している。今後それが続くと仮定することは、慎重であるべきだ」 "We have multiple return drivers, but honestly, most PE returns over the past decade were explained by multiple expansion. Assuming that continues demands caution."

IRRとMOICの算定 Calculating IRR and MOIC

LBO評価における2つの核心指標はMOIC（Money-on-Invested-Capital）とIRR（Internal Rate of Return）である。MOICは絶対的なリターン倍率を示し、IRRは時間価値を考慮したリターン率を示す。 The two core metrics in LBO analysis are MOIC (Money-on-Invested-Capital) and IRR (Internal Rate of Return). MOIC expresses the absolute return multiple; IRR expresses the return rate adjusted for the time value of money.

MOIC = 出口エクイティ手取り / エントリー・エクイティ投資額

IRR: エントリー・エクイティ × (1 + IRR)^n = 出口エクイティ手取り
→ IRR = (出口エクイティ / エントリー・エクイティ)^(1/n) − 1

例: エントリー・エクイティ 40、出口エクイティ 120、保有期間 5年
MOIC = 120 / 40 = 3.0×
IRR = (3.0)^(0.2) − 1 ≈ 24.6% MOIC = Exit Equity Proceeds / Entry Equity Investment

IRR: Entry Equity × (1 + IRR)^n = Exit Equity Proceeds
→ IRR = (Exit Equity / Entry Equity)^(1/n) − 1

e.g.: Entry Equity 40, Exit Equity 120, Hold Period 5 years
MOIC = 120 / 40 = 3.0×
IRR = (3.0)^(0.2) − 1 ≈ 24.6%

MOICとIRRは保有期間によって乖離する。MOIC 3.0×でも保有期間が3年なら IRR ≈ 44%、10年なら IRR ≈ 12%と大きく異なる。PE投資家がIRRを優先する背景には、ファンド存続期間（通常10年）の内でいかに資本効率を最大化するかという動機がある。 MOIC and IRR diverge depending on hold period. A 3.0× MOIC implies an IRR of ≈44% over 3 years but only ≈12% over 10 years. PE investors prioritise IRR because their mandate is to maximise capital efficiency within a fund's life (typically 10 years).

TRAP — 中間配当とIRR操作Dividend Recaps and IRR Gaming

保有期間中に配当（Dividend Recap）を実施することでエクイティ投資家への現金回収を早め、IRRを上昇させることができる。しかし追加デットによって企業のリスクプロファイルが悪化し、出口時の評価が下がる可能性がある。IRRの最大化が必ずしも企業価値の最大化と一致しない点は、LBO評価の本質的なエージェンシー問題の一つである。 A dividend recapitalisation during the hold period returns cash to equity investors earlier, boosting IRR. However, the additional debt deteriorates the company's risk profile and may compress exit valuation. IRR maximisation does not necessarily align with enterprise value maximisation — this tension is one of the fundamental agency problems in LBO structures.

デット・キャパシティの分析：負債は何が制約するか Debt Capacity Analysis: What Constrains the Debt Quantum

LBOにおいてデット水準の決定は、投資判断の中核を占める。デット水準が低すぎればIRRが低下し、高すぎれば財務的苦境（Financial Distress）のリスクを負う。実務では主に2つのアプローチが用いられる。 Determining the debt quantum is central to LBO investment decisions. Too little debt depresses IRR; too much creates financial distress risk. In practice, two primary approaches are used.

デット・キャパシティを決定する際、最も重要な変数はFCFの安定性である。EBITDAが高くてもCapexが重く、あるいは運転資本の季節変動が大きい事業は、見かけのEBITDAマルチプルより低いデット水準しか許容できない。対照的に、SaaSやフランチャイズのような予測可能な高FCFビジネスは、EBITDA倍率以上のデットを受け入れられることが多い。 The most critical variable in determining debt capacity is the stability of FCF. A business with high EBITDA but heavy capex or large seasonal working capital swings can support less debt than its EBITDA multiple implies. Conversely, highly predictable high-FCF businesses like SaaS or franchises often accommodate debt levels beyond what the EBITDA multiple alone would suggest.

資本構成の変化とWACC：なぜLBOにWACCベースのDCFが馴染まないか Changing Capital Structure and WACC: Why Constant-WACC DCF Is Ill-Suited to LBOs

通常のDCFはWACCを一定と仮定する。しかしLBOでは、エントリー時には高いレバレッジ（例：D/EV = 60%）が5年かけて低下する（例：D/EV = 20%）。このレバレッジの変化は理論的にWACCを年ごとに変化させるはずだ。一定WACCを使うことは、この動的な変化を無視する近似であり、特に初期年度のWACC過少評価につながりうる。 A standard DCF assumes a constant WACC. But in an LBO, leverage starts high at entry (e.g., D/EV = 60%) and declines over five years (e.g., D/EV = 20%). This structural change in leverage should theoretically change WACC each year. Using a constant WACC ignores this dynamic, systematically underestimating WACC in the high-leverage early years.

Miles-Ezzellモデル：レバレッジが変化する環境でのWACC計算には、Miles-Ezzellアプローチが理論的に整合する。各期のWACCを Ku − Kd × T × (D/V) で表し、毎年更新する。Kuはアンレバードコスト・オブ・キャピタル、Kdはデットコスト、TはTax Rate、D/Vは各期のレバレッジ比率。この計算の複雑さが、APVアプローチが選好される理由の一つである。 Miles-Ezzell Model: The Miles-Ezzell approach is theoretically consistent for WACC calculation in a changing-leverage environment. WACC for each period is expressed as Ku − Kd × T × (D/V), updated annually. Ku is the unlevered cost of capital, Kd is the cost of debt, T is the tax rate, and D/V is the leverage ratio in each period. The complexity of this calculation is one reason the APV approach is preferred.

出口マルチプルの感応度分析 Exit Multiple Sensitivity Analysis

LBOモデルにおいて、最大の不確実性を持つ変数は出口マルチプルである。エントリー・マルチプルは交渉で決まるが、出口マルチプルは5〜7年後の市場環境・事業パフォーマンス・IPO市場の状況に依存する。実務では出口EBITDA × 出口マルチプルの感応度テーブルをモデルの中心に置く。 In an LBO model, the variable carrying the greatest uncertainty is the exit multiple. The entry multiple is fixed in negotiation, but the exit multiple depends on market conditions, business performance, and IPO/M&A market conditions five to seven years hence. In practice, the sensitivity table of exit EBITDA × exit multiple is placed at the centre of the model.

Even with 20% EBITDA upside, an exit at 7× yields only 21% IRR; but with base EBITDA and an 11× exit, IRR reaches 30%. This explains why PE investors attach so much importance to macro and industry cycle timing.

Equity Waterfall：キャッシュの分配順序 Equity Waterfall: The Order of Cash Distribution

LBOの出口において、売却収入はある決まった順序で各資本提供者に分配される。これをEquity Waterfall（エクイティ・ウォーターフォール）と呼ぶ。分配の優先順位が低い者は、上位の請求権が満たされるまで手取りはゼロとなる。この非対称性がLBOの高レバレッジ戦略のリスク構造を規定する。 At exit, sale proceeds are distributed to capital providers in a defined sequence known as the equity waterfall. Those with lower priority receive nothing until superior claims are satisfied. This asymmetry defines the risk architecture of a high-leverage LBO strategy.

Equity Waterfallの一般的な順序Typical Equity Waterfall Sequence
① シニア・デット（銀行ローン・TLB等） Senior Debt (bank loans, TLB, etc.) 最優先First priority
② メザニン・劣後デット（ハイイールド債等） Mezzanine / Subordinated Debt (HY bonds, etc.) 第二優先Second priority
③ 優先株（Preferred Equity）— 累積優先配当あり Preferred Equity — with accrued preferred return 第三優先Third priority
④ 普通株（PEファンドのLP出資分） Common Equity (PE fund LP capital) 第四優先Fourth priority
⑤ マネジメント・スイート・エクイティ Management Sweet Equity 最後（ただしハードル超過後に急増）Last (but accelerates above hurdle)

マネジメント向けのスイート・エクイティは、一定のリターン閾値（ハードル・レート、例：1.5×MOIC）を超えた場合にのみ大きなリターンが発生するように設計されることが多い。これはエグゼクティブのインセンティブをPEファンドのリターン目標に整合させるメカニズムだ。ラチェット構造（Ratchet）を使えば、達成されたIRRに応じてマネジメントの持分割合が段階的に上昇するように設定できる。 Sweet equity for management is typically structured so that significant returns only accrue once a return threshold (hurdle rate, e.g., 1.5× MOIC) is exceeded. This mechanism aligns executive incentives with PE fund return targets. Using a ratchet structure, management's ownership percentage can be set to increase in steps as achieved IRR rises.

LBOとDCFの統合：APVアプローチ Integrating LBO with DCF: The APV Approach

LBOバリュエーションとDCFを理論的に統合する最も適切な枠組みはAPV（Adjusted Present Value）アプローチである。APVは企業価値を「アンレバードNPV」と「ファイナンシングの副次効果」に分解する。 The most theoretically appropriate framework for integrating LBO valuation with DCF is the APV (Adjusted Present Value) approach. APV decomposes enterprise value into "unlevered NPV" and the "side effects of financing."

APV = アンレバードNPV + PV(節税効果) + PV(財務的苦境コスト)

アンレバードNPV = FCF₁/(1+Ku) + FCF₂/(1+Ku)² + ... + TV/(1+Ku)ⁿ
（Ku = アンレバード・コスト・オブ・キャピタル）

PV(節税効果) = Σ[ Kd × T × Dt / (1+Ku)^t ]
（Dt = t年の残余デット、T = 実効税率）

PV(財務的苦境コスト) = 財務苦境の期待コスト × 苦境に陥る確率（通常は負の値） APV = Unlevered NPV + PV(Tax Shield) + PV(Financial Distress Costs)

Unlevered NPV = FCF₁/(1+Ku) + FCF₂/(1+Ku)² + ... + TV/(1+Ku)ⁿ
(Ku = unlevered cost of capital)

PV(Tax Shield) = Σ[ Kd × T × Dt / (1+Ku)^t ]
(Dt = remaining debt at year t, T = effective tax rate)

PV(Financial Distress Costs) = Expected cost of distress × Probability of distress (typically negative)

APVがLBOに適している理由は、デット残高が毎年変化しても各年の節税効果を個別に割り引けるからだ。WACCベースのDCFでは、節税効果は「レバレッジの変化に応じて変動するWACC」の中に暗黙的に含まれ、独立して可視化されない。APVは「レバレッジがどの程度、いつ、企業価値に貢献するか」を透明に示す点で優れる。 APV is well-suited to LBOs because it allows the tax shield from each year's debt balance to be discounted individually, even as the debt balance changes year by year. In a WACC-based DCF, the tax benefit is implicitly embedded in a WACC that would need to be updated annually — it is never made visible independently. APV is superior in transparently showing how much leverage contributes to value, and when.

#008との接続：前稿でコントロール・プレミアムとシナジーの評価にAPVアプローチへの言及があった。APVにおける財務シナジー（節税効果の増大・負債容量の拡大）の評価は、LBOのAPV計算と同一の理論構造を持つ。M&AにおけるLBO買収者が財務シナジーをどう評価するかは、APVの節税効果計算をそのまま適用することで導出できる。 Connection to #008: The previous article mentioned the APV approach for valuing financial synergies in M&A. The evaluation of financial synergies in APV — expanded tax shield, increased debt capacity — shares exactly the same theoretical structure as the APV calculation in an LBO. A strategic buyer acquiring a target through an LBO can apply APV's tax shield calculation directly to value financial synergies.

実務的なLBOモデルの構成と落とし穴 Practical LBO Model Structure and Common Pitfalls

標準的なLBOモデルは以下のモジュールで構成される。各モジュールの接続ミスが実務上の誤りの大半を占める。 A standard LBO model consists of the following modules. Errors in the connections between modules account for the majority of real-world modelling mistakes.

  • Sources & Uses Table：エントリーEV、デット構成（シニア／メザニン）、エクイティ出資額、取引費用の一覧 Sources & Uses Table: Entry EV, debt structure (senior/mezzanine), equity contribution, transaction fees
  • Income Statement Projection：売上→EBITDA→EBIT→EBT→Net Income の5年予測（インタレストは資本構成から逆算） Income Statement Projection: 5-year forecast from revenue to net income (interest back-calculated from capital structure)
  • Debt Schedule：各トランシュの元本・利払い・強制返済・スイープ（超過FCFによる繰上返済）のモデル Debt Schedule: Principal, interest, mandatory amortisation, and cash sweep (excess FCF prepayment) for each tranche
  • Cash Flow Statement：EBITDA→FCF→デット返済→Ending Cash の橋渡し計算（これが誤ると循環参照が発生する） Cash Flow Statement: EBITDA → FCF → debt repayment → ending cash bridge (errors here typically cause circular references)
  • Exit Analysis：出口EV（出口EBITDA × 出口マルチプル）→ Net Debt控除 → エクイティ → MOIC / IRR の計算 Exit Analysis: Exit EV (exit EBITDA × exit multiple) → deduct net debt → equity proceeds → MOIC / IRR calculation
  • Sensitivity Tables：出口マルチプル × 出口EBITDA の2軸感応度（IRRおよびMOIC） Sensitivity Tables: Two-axis sensitivity (IRR and MOIC) across exit multiple × exit EBITDA
TRAP — 循環参照の罠The Circular Reference Trap

デット残高 → 利払い → Net Income → FCF → デット返済 → デット残高 という回路が閉じることで、スプレッドシートに循環参照が発生しやすい。これを回避するために、多くのモデルはDebt Scheduleで「前期末デット残高」を利払い計算の基礎に使う。あるいはExcel の反復計算を有効にするが、これはモデルの検証を困難にするため、プロは基本的に前期残高ベースを選好する。 The circuit: debt balance → interest → net income → FCF → debt repayment → debt balance easily creates a circular reference in spreadsheet models. To avoid this, most models use end-of-prior-period debt as the basis for interest calculation in the Debt Schedule. Alternatively, Excel's iterative calculation can be enabled, but this complicates model auditing — most professionals prefer the prior-period balance approach.
